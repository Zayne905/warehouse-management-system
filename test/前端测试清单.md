# 入库管理 — 前端手动测试清单

## 测试前提
1. MySQL 已启动
2. 已执行 `V2__phase2_schema.sql` 和 `V3__test_data.sql`
3. 后端已启动: `cd warehouse-backend && mvn spring-boot:run` (端口 8081)
4. 前端已启动: `cd warehouse-frontend && npm run dev` (端口 5173)
5. 浏览器打开 `http://localhost:5173`
